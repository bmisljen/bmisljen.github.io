Episync Android Sync

Android app for bi-directional phone to PC sync over Wifi

My partner and I got funding from an angel invester to develop an app that auto syncs files and folders from Android to PC. The app enabled a user to select one or many files to be auto synced to a computer over a local Wifi connection and saved to a directory of choice. The PC based app was written in Java to increase portability accross Windows, OS X and Linux operating systems. The sync also worked the other way around by selecting one or many PC folders to be auto synced over local Wifi to a specific directory on an Android smartphone.

  • Date: 2013 - 2015
  • Category: Android App Development

Ultimate Q&A

Django Web App for Questions and Answers

Ultimate Q&A is a Django based web app developed with Django, Bootstrap, JQuery, Ajax, CSS, HTML and Font Awesome icons in order to create a simple questions and answers app with questions ratings done using Ajax. It stores and updates questions and answers using a PostgreSQL backend database.
